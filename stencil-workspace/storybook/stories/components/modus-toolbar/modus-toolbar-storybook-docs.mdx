# Toolbar

The `modus-toolbar` component provides a customizable Toolbar that can be positioned at the bottom of the screen, other positions are a work in progress. It's designed to offer users quick access to common actions or tools.

### Default

<Story id="components-toolbar--default" height="100px" />

```html
<modus-toolbar>
  <modus-button>Button 1</modus-button>
  <modus-button>Button 2</modus-button>
  <modus-button>Button 3</modus-button>
</modus-toolbar>
```

### Icon Only

<Story id="components-toolbar--icon-only" height="100px" />

```html
<modus-toolbar>
  <modus-tooltip text="undo" position="top">
    <modus-button icon-only="undo"></modus-button>
  </modus-tooltip>
  <modus-tooltip text="redo" position="top">
    <modus-button icon-only="redo"></modus-button>
  </modus-tooltip>
  <modus-tooltip text="expand" position="top">
    <modus-button icon-only="expand"></modus-button>
  </modus-tooltip>
  <modus-tooltip text="collapse" position="top">
    <modus-button icon-only="collapse"></modus-button>
  </modus-tooltip>
  <modus-divider></modus-divider>
  <modus-tooltip text="window_dock_undock" position="top">
    <modus-button icon-only="window_dock_undock"></modus-button>
  </modus-tooltip>
</modus-toolbar>
```

## Properties

| Property    | Attribute    | Description                     | Type      | Default |
| ----------- | ------------ | ------------------------------- | --------- | ------- |
| `ariaLabel` | `aria-label` | The Toolbar's aria-label        | `string`  |         |
| `disabled`  | `disabled`   | Whether the Toolbar is disabled | `boolean` | `false` |

### Accessibility

- Toolbar gets role="toolbar"
- Toolbar gets an `aria-label` provided by the `aria-label` property input.
- Toolbar button gets an `aria-label` provided by the `aria-label` property input.
