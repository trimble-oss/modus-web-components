# Content Tree

---

[Modus Content Tree](https://modus.trimble.com/components/content-tree/) provides users with a way to navigate nested hierarchical information using a parent-child relationship model.
It is located on the left side of the screen and cannot be combined with Side Navigation or Toolbar.

#### Implementation Details

- Content tree can be created using the components `modus-tree-view` in the root and `modus-tree-view-item` for nodes.
- The `modus-tree-view-item` takes `nodeId` which should be a unique number and the `label` prop to display the node text.
- Clicking on collapse/expand button expands or collapse a tree view item and clicking on the node selects it.

### Properties

##### Modus Tree View

| Name                     | Description                                  | Type    | Options                          | Default Value | Required                                     |
| ------------------------ | -------------------------------------------- | ------- | -------------------------------- | ------------- | -------------------------------------------- |
| checkbox-selection       | Enables checkbox selection on each tree item | boolean |                                  |               |                                              |
| disabled                 | Disables each tree item                      | boolean |                                  |               |                                              |
| multi-checkbox-selection | Enables multiple checkbox selection          | boolean |                                  |               |                                              |
| multi-selection          | Enables multiple tree items selection        | boolean |                                  |               |                                              |
| size                     | The default size of all tree items           | string  | 'condensed', 'standard', 'large' | standard      | <span class="material-icons">&#10004;</span> |

##### Modus Tree View Item

| Name          | Description                                   | Type    | Options | Default Value | Required                                     |
| ------------- | --------------------------------------------- | ------- | ------- | ------------- | -------------------------------------------- |
| checked       | Checked state of the tree item                | boolean |         |               |                                              |
| disabled      | Disables the tree item                        | boolean |         |               |                                              |
| expanded      | Expanded state of the tree item               | boolean |         |               |                                              |
| indeterminate | Checkbox indeterminate state of the tree item | boolean |         |               |                                              |
| label         | Label for the tree item                       | string  |         |               | <span class="material-icons">&#10004;</span> |
| node-id       | Unique tree item identifier                   | string  |         |               | <span class="material-icons">&#10004;</span> |
| selected      | Selected state of the tree item               | boolean |         |               |                                              |

### DOM Events

##### Modus Tree View Item

| Name             | Description                                      | Emits   |
| ---------------- | ------------------------------------------------ | ------- |
| checkboxClick    | An event that fires on tree item checkbox click  | boolean |
| itemClick        | An event that fires on tree item click           | boolean |
| itemExpandToggle | An event that fires on tree item expand/collapse | boolean |
