# Checkbox

---

[Modus Checkbox](https://modus.trimble.com/components/checkboxes/) web components are wrappers around native `<input type="checkbox">` elements. They are referenced using the `<modus-checkbox>` custom HTML element.

This component is compatible with Angular reactive forms. This can be achieved through the implementation of a simple directive.

### Accessibility

- Checkbox gets an `aria-checked` set to whether Checkbox is checked.
- Checkbox gets an `aria-label` provided by the `aria-label` property input.
- Checkbox gets an `aria-disabled` set to whether Checkbox is disabled.
- When Checkbox has focus, **Space** or **Enter** toggle it, and emit `checkboxClick` event.
