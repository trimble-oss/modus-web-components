# Icons

---

[Modus Icon](https://modus-icons.trimble.com/modus-solid/) Modus Icon component is a wrapper for icons that are replicated from https://modus-icons.trimble.com. It is referenced using the `<modus-icon>` custom HTML element.

### Default

<modus-icon name="sun" color="black" size="50"></modus-icon>
<modus-icon name="sun" color="black" size="40"></modus-icon>
<modus-icon name="sun" color="black" size="25"></modus-icon>
<modus-icon name="sun" color="black" size="15"></modus-icon>

```html
<modus-icon name="sun" color="black" size="50"></modus-icon>
<modus-icon name="sun" color="black" size="40"></modus-icon>
<modus-icon name="sun" color="black" size="25"></modus-icon>
<modus-icon name="sun" color="black" size="15"></modus-icon>
```

### Types
```ts
type ModusIconName = /* View https://modus-icons.trimble.com/modus-solid/ for a full list of names */
```


### Properties

| Property | Attribute | Description                      | Type            | Default     |
| -------- | --------- | -------------------------------- | --------------- | ----------- |
| `color`  | `color`   | (optional) The color of the Icon | `string`        | `undefined` |
| `name`   | `name`    | The name of the icon             | `ModusIconName` | `undefined` |
| `size`   | `size`    | (optional) The size of the Icon  | `string`        | `'16'`      |

### DOM Events

| Name     | Description                      | Emits |
| -------- | -------------------------------- | ----- |
| `iconClick`| Fires when the icon is clicked   |       |
