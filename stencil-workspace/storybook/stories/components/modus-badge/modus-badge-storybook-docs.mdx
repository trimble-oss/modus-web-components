# Badge

---

[Modus Badge](https://modus.trimble.com/components/badges/) web components provide non-interactive labels for displaying small amounts of information. They are referenced using the `<modus-badge>` custom HTML element.
This component utilizes the slot element, allowing you to render your own HTML in the badge.

### Accessibility

- Badge has `role` of `status`.
- Badge gets an `aria-label` provided by the `aria-label` property input.
