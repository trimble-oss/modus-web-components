# List

---

[Modus List](https://modus.trimble.com/components/lists/) web components are wrappers around native `<modus-list-item>` custom HTML elements. They are referenced using the `<modus-list>` custom HTML element.

The `<modus-list-item>` utilizes the slot element, allowing you to render your own HTML in the component.

#### Implementation Details

- To support a wide variety of selection methods (single, multi, etc) it is up to the implementation to control which items are selected.

### Default

<modus-list>
  <modus-list-item size="condensed">Condensed</modus-list-item>
  <modus-list-item selected size="condensed">
    Condensed & Selected
  </modus-list-item>
  <modus-list-item>Default</modus-list-item>
  <modus-list-item selected>Selected</modus-list-item>
  <modus-list-item disabled>Disabled</modus-list-item>
  <modus-list-item size="large">Large</modus-list-item>
  <modus-list-item selected size="large">
    Large & Selected
  </modus-list-item>
</modus-list>

```html
<modus-list>
  <modus-list-item size="condensed">Condensed</modus-list-item>
  <modus-list-item selected size="condensed"
    >Condensed & Selected</modus-list-item
  >
  <modus-list-item>Default</modus-list-item>
  <modus-list-item selected>Selected</modus-list-item>
  <modus-list-item disabled>Disabled</modus-list-item>
  <modus-list-item size="large">Large</modus-list-item>

</modus-list>
```
