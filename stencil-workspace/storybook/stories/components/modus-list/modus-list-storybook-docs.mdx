# List

---

[Modus List](https://modus.trimble.com/components/lists/) web components are wrappers around native `<modus-list-item>` custom HTML elements. They are referenced using the `<modus-list>` custom HTML element.

The `<modus-list-item>` utilizes the slot element, allowing you to render your own HTML in the component.

#### Implementation Details

- To support a wide variety of selection methods (single, multi, etc) it is up to the implementation to control which items are selected.

### Properties

##### Modus List Item

| Property     | Attribute    | Description                                          | Type                                   | Default      |
| ------------ | ------------ | ---------------------------------------------------- | -------------------------------------- | ------------ |
| `borderless` | `borderless` | (optional) Whether the list item has a border or not | `boolean`                              | `undefined`  |
| `disabled`   | `disabled`   | (optional) Disables the list item                    | `boolean`                              | `undefined`  |
| `selected`   | `selected`   | (optional) The selected state of the list item       | `boolean`                              | `undefined`  |
| `size`       | `size`       | (optional) The size of list item                     | `"condensed", "large", "standard"` | `'standard'` |
| `type`       | `type`       | (optional) The type of list item                     | `string`                               | `'standard'` |


### DOM Events

##### Modus List Item

| Event       | Description                            | Type               |
| ----------- | -------------------------------------- | ------------------ |
| `itemClick` | An event that fires on list item click | `CustomEvent<any>` |

### Methods

##### Modus List Item

| Method name  | Description                     | Parameter          | Return            |
| ------------ | ------------------------------- | ------------------ | ----------------- | ----- |
| `focusItem`    | Focus the list item      | `Promise<void>`       | `Promise<void>`   |

### Shadow Parts

| Part           | Description                                                              |
| -------------- | ------------------------------------------------------------------------ |
| `"list-items"` | To apply a custom style to the `ul` element in `modus-list`'s shadow DOM |
