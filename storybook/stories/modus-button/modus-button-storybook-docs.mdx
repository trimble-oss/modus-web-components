# Button
---

[Modus Button](https://modus.trimble.com/elements/buttons/) web components are wrappers around native `<button>` elements. They are referenced using the `<modus-button>` custom HTML element.
This component utilizes the slot element, allowing you to render your own HTML in the button.

Implementation Details
- Outline buttons only support 'primary' and 'secondary' colors.
- Borderless buttons only support a single color.

### Accessibility
- Button has `role` of `button`.
- Button gets an `aria-label` provided by the `aria-label` property input.
- Button gets an `aria-disabled` set to whether Button is disabled.
- Button gets an `aria-pressed` set to whether Button is pressed.
- When Button has focus, **Space** or **Enter** activate it.
