# Text Input
---

[Modus Text](https://modus.trimble.com/elements/inputs/) Input web components are wrappers around native `<input type="text">` elements. They are referenced using the `<modus-text-input>` custom HTML element.

This component is compatible with Angular reactive forms. This can be achieved through the implementation of a simple directive.

### Accessibility
- Text Input gets an `aria-label` provided by the `aria-label` property input.
- Text Input gets an `aria-disabled` set to whether Text Input is disabled.
- Text Input gets an `aria-placeholder` provided by the `placeholder` property input.
- Text Input gets an `aria-invalid` set to whether Text Input is invalid.
- Text Input gets an `aria-readonly` set to whether Text Input is readonly.
- Text Input gets an `aria-required` set to whether Text Input is required.
