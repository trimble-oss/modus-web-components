### Accessibility

- Switch has `role` of `switch`.
- Switch gets an `aria-checked` set to whether Switch is checked.
- Switch gets an `aria-label` provided by the `aria-label` property input.
- Switch gets an `aria-disabled` set to whether Switch is disabled.
- When Switch has focus, **Space** or **Enter** toggle it, and emits the `switchClick` event.
