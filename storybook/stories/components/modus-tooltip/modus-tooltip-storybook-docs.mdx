# Tooltip

---

[Modus Tooltip](https://modus.trimble.com/components/tooltips/) web components display the completion progress of a task. They are referenced using the `<modus-tooltip>` custom HTML element.
This component utilizes the slot element, allowing you to wrap anything with a tooltip.

### Default

<modus-tooltip text="Tooltip text..." position="right">
  <modus-button>Button</modus-button>
</modus-tooltip>

```html
<modus-tooltip text="Tooltip text..." position="right">
  <modus-button>Button</modus-button>
</modus-tooltip>
```

### Accessibility

- Tooltip has `role` of `tooltip`.
- Tooltip gets an `aria-label` provided by the `aria-label` property input.
