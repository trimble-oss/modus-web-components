# Progress Bar
---

[Modus Progress Bar](https://modus.trimble.com/elements/progress-bars/) web components display the completion progress of a task. They are referenced using the `<modus-progress-bar>` custom HTML element.
### Default
<div>
  <modus-progress-bar value="3" max-value="4" min-value="0" text="75%"></modus-progress-bar>
</div>

```html
<modus-progress-bar value="3" max-value="4" min-value="0" text="75%"></modus-progress-bar>
```

### Accessibility
- Progress Bar has `role` of `progressbar`.
- Progress Bar gets an `aria-label` provided by the `aria-label` property input.
- Progress Bar gets an `aria-maxvalue` provided by the `max-value` property input.
- Progress Bar gets an `aria-minvalue` provided by the `min-value` property input.
- Progress Bar gets an `aria-valuenow` set to the current value.
