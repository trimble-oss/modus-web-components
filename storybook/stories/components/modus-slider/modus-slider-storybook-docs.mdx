# Slider

---

[Modus Slider](https://modus.trimble.com/components/sliders/) web components are referenced using the `<modus-slider>` custom HTML element.

### Default

<modus-slider label="Slider"></modus-slider>

<modus-slider disabled label="Disabled slider"></modus-slider>

```html
<modus-slider label="Slider"></modus-slider> <modus-slider disabled label="Disabled slider"></modus-slider>
```

### Accessibility

- Slider gets an `aria-label` provided by the `aria-label` property input.
- Slider gets an `aria-disabled` set to whether Slider is disabled.
- Slider gets an `aria-valuemax` set to Slider's maximum value.
- Slider gets an `aria-valuemin` set to Slider's minimum value.
- Slider gets an `aria-valuenow` set to Slider's current value.
