# Spinner

---

[Modus Spinner](https://modus.trimble.com/components/spinners/) web components indicate progress by showing a loading state. They are referenced using the `<modus-spinner>` custom HTML element.

### Default

<modus-spinner></modus-spinner>

<modus-button color="primary">
  <modus-spinner color="white" size=".5rem"></modus-spinner>
  Loading...
</modus-button>

```html
<!-- Spinner -->
<modus-spinner></modus-spinner>

<!-- Render in another element with different color and size -->
<modus-button color="primary">
  <modus-spinner color="white" size=".5rem"></modus-spinner>
  Loading...
</modus-button>
```

### Properties

| Name    | Description              | Type     | Options | Default Value                | Required |
| ------- | ------------------------ | -------- | ------- | ---------------------------- | -------- |
| `color` | The color of the spinner | `string` |         | `#005F9E` (Trimble Blue Mid) |          |
| `size`  | The size of the spinner  | `string` |         | '2rem'                       |          |

### Accessibility

- Spinner has an `aria-busy` attribute that is set to `true`.
