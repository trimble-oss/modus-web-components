# Alert
---

[Modus Alert](https://modus.trimble.com/elements/alerts/) web components provide well defined elements for rendering system level messages. They are referenced using the `<modus-alert>` custom HTML element.

<Anchor storyId="components-alert--default" />

### Default
<modus-alert message="Info alert (default)"></modus-alert>


<Anchor storyId="components-alert--dismissible" />

### Dismissible
<modus-alert dismissible message="Dismissible alert"></modus-alert>


<Anchor storyId="components-alert--error" />

### Error
<modus-alert message="Error alert" type="error"></modus-alert>

<Anchor storyId="components-alert--info-gray" />

### Info Gray
<modus-alert message="Info gray alert" type="info-gray"></modus-alert>


<Anchor storyId="components-alert--info-gray-dark" />

### Gray Dark
<modus-alert message="Info gray dark alert" type="info-gray-dark"></modus-alert>


<Anchor storyId="components-alert--success" />

### Success
<modus-alert message="Success alert" type="success"></modus-alert>


<Anchor storyId="components-alert--warning" />

### Warning
<modus-alert message="Warning alert" type="warning"></modus-alert>


```html
<modus-alert message="Info alert (default)"></modus-alert>
<modus-alert dismissible message="Dismissible alert"></modus-alert>
<modus-alert message="Error alert" type="error"></modus-alert>
<modus-alert message="Info gray alert" type="info-gray"></modus-alert>
<modus-alert message="Info gray dark alert" type="info-gray-dark"></modus-alert>
<modus-alert message="Success alert" type="success"></modus-alert>
<modus-alert message="Warning alert" type="warning"></modus-alert>
```
