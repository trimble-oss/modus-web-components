# Breadcrumb
---

[Modus Breadcrumb](https://modus.trimble.com/elements/breadcrumbs/) web components are wrappers around native `<nav>` elements. They are referenced using the `<modus-breadcrumb>` custom HTML element.

A new TypeScript typing has been provided named Crumb defined as:

    {
      id: string; // required, must be unique.
      display: string; // required.
      route?: string; // optional, leave undefined to handle routing externally.
    }

Implementation Details
- Crumb id values must be unique within the crumbs collection.
- To handle routing outside of the breadcrumbs, leave the route value undefined and listen to the crumbClick DOM event.

### Default
<Story id="components-breadcrumb--default" />

```html
<modus-breadcrumb></modus-breadcrumb>

<script>
    const modusBreadcrumb = document.querySelector('modus-breadcrumb');
    const crumbs = [
        { id: '1', display: 'Crumb 1' },
        { id: '2', display: 'Crumb 2' },
        { id: '3', display: 'Crumb 3' },
        { id: '4', display: 'Crumb 4' },
    ];
    modusBreadcrumb.crumbs = crumbs;
</script>
```

### Accessibility
- Breadcrumb has `role` of `navigation`.
- When a Breadcrumb has focus, **Enter** emits the `crumbClick` event.
