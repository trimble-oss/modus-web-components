# Pagination
---

[Modus Pagination](https://modus.trimble.com/components/pagination/) web components are referenced using the `<modus-pagination>` custom HTML element.

<modus-pagination active-page="8" max-page="32" min-page="1" size="large"></modus-pagination>

<modus-pagination active-page="8" max-page="32" min-page="1" size="large"></modus-pagination>
