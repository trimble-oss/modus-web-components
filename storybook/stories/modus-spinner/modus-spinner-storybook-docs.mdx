# Spinner
---

[Modus Spinner](https://modus.trimble.com/elements/spinners/) web components indicate progress by showing a loading state. They are referenced using the `<modus-spinner>` custom HTML element.

### Default

<modus-spinner></modus-spinner>

<modus-button color="primary">
  <modus-spinner color="white" size=".5rem"></modus-spinner>
  Loading...
</modus-button>

```html
<!-- Spinner -->
<modus-spinner></modus-spinner>

<!-- Render in another element with different color and size -->
<modus-button color="primary">
    <modus-spinner color="white" size=".5rem"></modus-spinner>
    Loading...
</modus-button>
```

- Spinner has an `aria-busy` attribute that is set to `true`.
